Learning Git

A quick look into learning Git on Windows

To get to grips with Git I downloaded Git Bash, a way to access Git through command line on windows PC's, with this I created a folder called MooseInvaders. This is where all the stuff I upload will go to, the game and the complimentary site I built alongside it.

I used several commands to set up my Git Repo and these were $git init to start up the git repo locally then $git add . which added all the files that were currently in the folder I initialised git in.

Once all the file were added the next step was to check the status $git status which showed if files needed to be committed, which all files were so I committed the files $git commit -am "first commit" this locally committed all files to the repo

What next? Uploading to the online repo on github itself

$git remote add origin https://github.com/moosehughes/MooseInvaders.git and then $git push -u origin master which pushes all file into your online repository.
