Sprite Sheets

And How they work inside Canvas

We start off by Looking at Cykod's Alien Invaders Github Repo that was downloaded to reverse engineer and glance over the code to begin with and decide to start on the sprites and how they work with the game itself. The original sprite sheet shows basic sprites for the game Cykod created


Now lets see how these work in the game...

This image shows the game gathering the sprite data and where the sprites are positioned on the sheet. It then defines a sprite variable using these positionings, telling the game to only use those certain areas of the sheet for that one sprite.


Here we have the engine.js calling the array of sprite variables from the image before and loading them into the game
